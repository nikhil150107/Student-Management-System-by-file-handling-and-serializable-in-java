# Student Management System

A console-based Student Management System built in Java.

## Features
- Add Student
- View All Students
- Search Student by ID
- Update Student
- Delete Student
- Data persists using Java Serialization (.ser file)

## Technologies Used
- Java
- File Handling
- Java Serialization (ObjectInputStream / ObjectOutputStream)

## How to Run
1. Clone the repo
2. Open in Eclipse IDE
3. Right-click `studentproject.java` → Run As → Java Application
